# Portafolio de Evidencias — Cómputo en la Nube

Portafolio académico desarrollado para la materia de Cómputo en la Nube del Instituto Tecnológico de Saltillo (TecNM). Documenta las actividades, prácticas y aprendizajes del semestre Enero–Junio 2026.

---

## ¿Qué contiene?

Un sitio web SPA organizado por unidades. Cada unidad es expandible y muestra sus actividades con descripción, resultados, reflexión personal y visor del PDF correspondiente. Incluye portada, introducción, objetivos, reflexión final y conclusión.

---

## Unidades

- **Unidad 1** — Fundamentos y conceptos del cómputo en la nube (4 actividades)
- **Unidad 2** — Exploración de plataformas: AWS, Google Cloud, Azure y GitHub (4 actividades)
- **Unidad 3** — Microservicios y flujo DevOps (1 actividad)
- **Unidad 4** — Ciberseguridad en la nube (9 actividades)
- **Unidad 5** — Implementación: VMs, Docker, LAMP, CRUD y migraciones (7 actividades)

---

## Stack

- React 19 + Vite
- Tailwind CSS
- React Router DOM
- Material Symbols (Google Icons)

---

## Instalación

```bash
npm install
npm run dev
```

Build de producción:

```bash
npm run build
```

---

## Agregar evidencias

Edita `src/data/evidencias.js`. Cada objeto tiene:

```js
{
  id: "1.1",
  unidad: 1,
  titulo: "...",
  pdf: "/pdfs/unidad1/archivo.pdf",
  tipo: "Práctica",
  descripcion: "...",
  fecha: "6 de febrero de 2026",
  resultados: "...",
  reflexion: "...",
  anexos: "..."
}
```

Los PDFs van en `public/pdfs/unidadN/` con nombres en minúsculas sin espacios.
